# List Sections

This lists all the sections for the forum.

**URL**: `/section/all`

**Method**: `GET`

**Auth Required**: NO

## Success Response

**Code**: `200 OK`

**Content Example**

```json
[
    {
        "id": "randomId",
        "name": "name",
        "order": "orderNumber",
        "updated": "update time",
        "created": "created time"
    },
    {
        "id": "randomId",
        "name": "name",
        "order": "orderNumber",
        "updated": "update time",
        "created": "created time"
    }
]
```
